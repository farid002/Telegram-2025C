# Telegram Bot Workshop 3 - Sənədləşmə

Bu workshop 6 müxtəlif Telegram bot layihəsindən ibarətdir. Hər tələbə üçün tam müstəqil və funksional bot hazırlanmışdır.

## Workshop Strukturu

Bu workshop-da 6 müxtəlif bot layihəsi var:

1. **Student 1: X-O Oyun Botu** - Tic-Tac-Toe oyunu ilə AI rəqib
2. **Student 2: Gündəlik Vərdiş İzləyici Bot** - Vərdiş izləmə və statistika
3. **Student 3: Viktorina Master Bot** - Müxtəlif mövzularda viktorinalar
4. **Student 4: Adam Asma Oyunu Bot** - Klassik hangman oyunu
5. **Student 5: Xərclər İzləyici Bot** - Şəxsi maliyyə idarəetməsi
6. **Student 6: Söz Tapmacası Bot** - Anagram və söz oyunları

## Ümumi Tələblər

Bütün botlar üçün eyni texnologiyalar istifadə olunur:

- **Python 3.8+** - Proqramlaşdırma dili
- **python-telegram-bot (v20+)** - Telegram Bot API
- **SQLite3** - Verilənlər bazası
- **python-dotenv** - Konfiqurasiya idarəetməsi

## Hər Tələbə üçün Sənədləşmə

Hər tələbə üçün aşağıdakı sənədlər mövcuddur:

- **README.md** - Layihənin məqsədi və ümumi məlumat
- **setup_guide.md** - Addım-addım qurulum təlimatı
- **technical_docs.md** - Texniki detallar və kod strukturu

## Ümumi Telegram Bot Qurulumu

Bütün botlar üçün Telegram bot qurulumu eynidir. Ətraflı məlumat üçün [telegram_bot_setup.md](telegram_bot_setup.md) faylına baxın.

## Növbəti Addımlar

1. Hər tələbə öz botunun qovluğuna keçir
2. `setup_guide.md` faylını oxuyur
3. Botu quraşdırır və işə salır
4. `technical_docs.md` faylından texniki detalları öyrənir
