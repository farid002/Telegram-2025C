# Student 3: Viktorina Master Bot

## Layihə Məqsədi

Bu layihədə müxtəlif mövzularda viktorinalar təklif edən Telegram bot hazırlayacaqsınız. Bot ilə biliklərinizi yoxlaya, xal toplaya və liderboard-da yer tuta biləcəksiniz.

## Nəticə

Layihəni tamamladıqdan sonra:
- Müxtəlif kateqoriyalarda viktorinalar oynaya biləcəksiniz
- Xal toplaya biləcəksiniz
- Liderboard-da yer tuta biləcəksiniz
- Statistikalarınızı görə biləcəksiniz

## Əsas Xüsusiyyətlər

1. **Müxtəlif Kateqoriyalar** - Riyaziyyat, tarix, elm, ədəbiyyat, idman, coğrafiya
2. **10 Sual** - Hər viktorinada 10 sual
3. **Xal Sistemi** - Hər düzgün cavab üçün 1 xal
4. **Liderboard** - Ən yaxşı oyunçular
5. **Statistika** - Şəxsi nəticələr

## Texniki Komponentlər

- **questions.py** - Sual bazası
- **quiz_engine.py** - Viktorina mühərriki
- **database.py** - Verilənlər bazası
- **bot.py** - Telegram bot

## İstifadə Səhnələri

1. İstifadəçi kateqoriya seçir
2. 10 sual cavablandırır
3. Xal toplayır
4. Nəticə görür və statistika yenilənir
