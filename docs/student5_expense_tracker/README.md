# Student 5: Xərclər İzləyici Bot

## Layihə Məqsədi

Bu layihədə şəxsi maliyyənizi idarə edə biləcəyiniz Telegram bot hazırlayacaqsınız. Bot ilə xərclərinizi və gəlirlərinizi qeyd edə, hesabatlar görə biləcəksiniz.

## Nəticə

Layihəni tamamladıqdan sonra:
- Xərc və gəlir qeydiyyatı edə biləcəksiniz
- Kateqoriyalar üzrə izləmə görə biləcəksiniz
- Günlük və aylıq hesabatlar ala biləcəksiniz
- Büdcə təyin edə biləcəksiniz

## Əsas Xüsusiyyətlər

1. **Xərc Qeydiyyatı** - Kateqoriyalarla xərc əlavə etmə
2. **Gəlir Qeydiyyatı** - Gəlir əlavə etmə
3. **Hesabatlar** - Günlük və aylıq hesabatlar
4. **Büdcə İzləməsi** - Büdcə vəziyyəti
5. **Balans Hesablaması** - Gəlir - Xərc

## Texniki Komponentlər

- **expense_manager.py** - Xərc idarəetməsi
- **reports.py** - Hesabatlar
- **database.py** - Verilənlər bazası
- **bot.py** - Telegram bot
