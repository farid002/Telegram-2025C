# Student 4: Adam Asma Oyunu Bot

## Layihə Məqsədi

Bu layihədə klassik Adam Asma oyununu Telegram üzərində oynaya biləcəyiniz bot hazırlayacaqsınız. Bot müxtəlif çətinlik səviyyələri və kateqoriyalar təklif edəcək.

## Nəticə

Layihəni tamamladıqdan sonra:
- Adam Asma oyunu oynaya biləcəksiniz
- Müxtəlif çətinlik səviyyələrini seçə biləcəksiniz
- Vizual oyun göstəricisi görə biləcəksiniz
- Statistikalarınızı izləyə biləcəksiniz

## Əsas Xüsusiyyətlər

1. **Çətinlik Səviyyələri** - Asan, orta, çətin
2. **Kateqoriyalar** - Heyvanlar, şəhərlər, meyvələr, idman
3. **Vizual Göstərici** - Adamın vəziyyəti
4. **Xal Sistemi** - Streak izləməsi
5. **Azərbaycan Əlifbası** - Azərbaycan hərfləri ilə

## Texniki Komponentlər

- **word_database.py** - Söz bazası
- **game_logic.py** - Oyun məntiqi
- **database.py** - Verilənlər bazası
- **bot.py** - Telegram bot
