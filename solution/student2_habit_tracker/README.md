# Gündəlik Vərdiş İzləyici Bot

Telegram üzərində gündəlik vərdişlərinizi izləyə biləcəyiniz bot.

## Quraşdırma

1. `pip install -r requirements.txt`
2. `.env.example` faylını `.env` adı ilə kopyalayın
3. `.env` faylına BOT_TOKEN əlavə edin
4. `python bot.py` ilə botu işə salın

## İstifadə

- `/start` - Botu başlat
- `/addhabit <ad>` - Yeni vərdiş əlavə et
- `/myhabits` - Vərdişlərinizi görün
- `/stats` - Statistikaları görün
