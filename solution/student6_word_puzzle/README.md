# Söz Tapmacası Bot

Telegram üzərində müxtəlif söz oyunları oynaya biləcəyiniz bot.

## Quraşdırma

1. `pip install -r requirements.txt`
2. `.env.example` faylını `.env` adı ilə kopyalayın
3. `.env` faylına BOT_TOKEN əlavə edin
4. `python bot.py` ilə botu işə salın

## İstifadə

- `/start` - Botu başlat
- `/anagram <hərflər>` - Anagram həll et
- `/puzzle` - Tapmaca oyna
- `/daily` - Gündəlik tapmaca
- `/stats` - Statistikaları görün
