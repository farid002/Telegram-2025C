# X-O Oyun Botu

Telegram üzərində X-O (Tic-Tac-Toe) oyunu oynaya biləcəyiniz bot.

## Quraşdırma

1. `pip install -r requirements.txt`
2. `.env.example` faylını `.env` adı ilə kopyalayın
3. `.env` faylına BOT_TOKEN əlavə edin
4. `python bot.py` ilə botu işə salın

## İstifadə

- `/start` - Botu başlat
- `/newgame` - Yeni oyun başlat
- `/stats` - Statistikaları görün
