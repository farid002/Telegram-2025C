# Xərclər İzləyici Bot

Telegram üzərində şəxsi maliyyənizi idarə edə biləcəyiniz bot.

## Quraşdırma

1. `pip install -r requirements.txt`
2. `.env.example` faylını `.env` adı ilə kopyalayın
3. `.env` faylına BOT_TOKEN əlavə edin
4. `python bot.py` ilə botu işə salın

## İstifadə

- `/start` - Botu başlat
- `/addexpense <məbləğ>` - Xərc əlavə et
- `/addincome <məbləğ>` - Gəlir əlavə et
- `/report` - Hesabat görün
